# What is an ENSIP?

An ENS Improvement Proposal (ENSIP) is a design document that describes a new feature, standard, or process for the ENS protocol and ecosystem. ENSIPs are the primary mechanism for proposing new features, collecting community input, and documenting design decisions.

This standalone site currently tracks one proposal:

- **[ENSIP-27: Node Classification and Metadata](/ensip/27)** &mdash; a draft standard for attaching structured, machine-readable metadata to ENS names and subnames, currently under review in [ensdomains/ensips#64](https://github.com/ensdomains/ensips/pull/64).

The canonical list of all ENSIPs lives at [docs.ens.domains/ensip](https://docs.ens.domains/ensip).
